Task : Write a query to count the number of students across departments who has scored more than 80 marks.
Alias the count column as 'Dept_HighScore_Count' Your table 'student' has the following columns:

St_id
St_name
Marks
Department

Code : select Department , SUM(case when marks>80 then 1 else 0 end) as Dept_HighScore_Count
from student group by department
